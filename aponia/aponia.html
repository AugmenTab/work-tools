How to Use This Tool

Click on the department you wish to see notes for.

Enter the code of the shop you wish to search into the text field (the code for this demo is P0574).

No need to worry about capitalizing - the tool is case-insensitive.

Once the tool has found an exact match, it will display automatically.

The tool will automatically reset after five minutes of inactivity.
